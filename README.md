# Книжный клуб колледжа Singularity

Минималистичный одностраничный сайт книжного клуба колледжа Singularity. Проект помогает участникам клуба смотреть книги текущего семестра, читать и добавлять отзывы, находить театральные события в Чебоксарах и предлагать книги или фильмы для будущих обсуждений.

## О проекте

Сайт создан как учебный фронтенд-проект без серверной части. Все данные, которые пользователь добавляет через формы, сохраняются локально в браузере с помощью `localStorage`.

Проект подходит для демонстрации навыков работы с HTML, CSS и JavaScript: семантическая разметка, адаптивная вёрстка, формы, валидация, фильтрация карточек, работа с DOM и локальным хранилищем.

## Основной функционал

* Главная секция с кратким описанием книжного клуба и информацией о следующей встрече.
* Блок «О клубе» с целью, форматом встреч и правилами участия.
* Каталог книг текущего семестра.
* Поиск по книгам по названию, автору, жанру, описанию и статусу.
* Кнопка «Добавить отзыв» у каждой книги, которая автоматически подставляет название книги в форму отзыва.
* Форма добавления отзывов с проверкой данных.
* Список отзывов с рейтингом в виде звёзд.
* Удаление одного отзыва или очистка всех отзывов.
* Афиша балетов в Чебоксарах.
* Форма рекомендаций книг и фильмов от студентов.
* Ограничение: с одного браузера можно оставить только одну рекомендацию в месяц.
* Список рекомендаций с возможностью удаления.
* Адаптивное мобильное меню.
* Сохранение отзывов и рекомендаций в браузере пользователя.

## Как запустить проект

1. Скачайте или склонируйте проект.
2. Убедитесь, что файлы находятся в одной папке:

```text
index.html
style.css
script.js
README.md
```

3. Откройте файл `index.html` в браузере.

Сайт не требует установки зависимостей, сборщика, сервера или подключения базы данных.

## Как пользоваться сайтом

### Просмотр книг

Перейдите в раздел «Книги». На странице отображаются книги текущего семестра:

* «Посёлок» — Кир Булычёв;
* «Пикник на обочине» — Аркадий и Борис Стругацкие;
* «Мы» — Евгений Замятин;
* «Записки юного врача + Морфий» — Михаил Булгаков;
* «Кэрри» — Стивен Кинг;
* «Овод» — Этель Лилиан Войнич.

Можно использовать строку поиска, чтобы найти книгу по названию, автору, жанру, описанию или статусу.

### Добавление отзыва

В разделе «Отзывы» пользователь может указать:

* имя;
* название книги, фильма или мероприятия;
* оценку от 1 до 5;
* текст отзыва.

После отправки отзыв появляется в списке и сохраняется в браузере.

### Добавление рекомендации

В разделе «Рекомендации» студент может предложить книгу или фильм. Нужно заполнить:

* имя;
* тип рекомендации;
* название;
* жанр;
* объяснение, почему это стоит обсудить.

После отправки рекомендация сохраняется в браузере. Повторная отправка в том же месяце с этого браузера блокируется.

### Театральная афиша

В разделе «Театр» отображаются балетные события в Чебоксарах, которые клуб может посетить и обсудить вместе.

## Работа с LocalStorage

Проект использует три ключа локального хранилища:

```text
singularityBookClubReviews
singularityBookClubRecommendations
singularityBookClubRecommendationMonth
```

Первый ключ хранит отзывы, второй — рекомендации, третий — месяц, в котором пользователь уже оставил рекомендацию.

Данные сохраняются только в браузере пользователя. Если открыть сайт на другом устройстве или в другом браузере, добавленные отзывы и рекомендации не появятся.

## Валидация форм

В форме отзывов проверяется:

* имя минимум из 2 символов;
* название минимум из 2 символов;
* выбранная оценка;
* текст отзыва минимум из 10 символов.

В форме рекомендаций проверяется:

* имя минимум из 2 символов;
* выбранный тип;
* название минимум из 2 символов;
* жанр минимум из 3 символов;
* описание минимум из 10 символов;
* отсутствие уже отправленной рекомендации в текущем месяце.

## Адаптивность

Сайт адаптирован под разные размеры экранов:

* на широких экранах используются многоколоночные сетки;
* на планшетах карточки перестраиваются в две колонки;
* на мобильных устройствах карточки становятся одноколоночными;
* навигация превращается в раскрывающееся меню.
